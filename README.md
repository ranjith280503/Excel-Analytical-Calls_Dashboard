# Calls Dashboard Using Excel     


## Project Overview
This project is an interactive Calls Dashboard developed in Microsoft Excel to analyze and visualize call center performance data. The dashboard helps users monitor key metrics, identify trends, and make data-driven decisions.

## Features
- Interactive Dashboard
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Dynamic KPIs
- Data Cleaning and Transformation
- Automated Calculations using Excel Formulas

## Data Analysis Techniques
- Data Cleaning
- Data Validation
- Sorting and Filtering
- Pivot Table Analysis
- Trend Analysis
- Performance Tracking

## Dashboard Components
1. KPI Cards
2. Call Volume Analysis
3. Agent Performance Analysis
4. Department-wise Call Analysis
5. Call Duration Analysis
6. Interactive Filters (Slicers)
7. Charts and Visualizations

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
